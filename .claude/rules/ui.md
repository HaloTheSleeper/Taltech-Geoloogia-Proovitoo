# UI & Styling Rules

## UI library

- Use shadcn-vue components when appropriate.
- Do not introduce other UI frameworks.

## Tailwind

- All colors, fonts and spacing must come from Tailwind theme.
- Do not hardcode random colors.

## Component rules

Components must:

- be presentational
- receive props (if needed)
- emit events (if needed)
- not fetch data directly (use composables for that)

## Design style

- Clean, minimal, readable.
- Consistent spacing and typography.
